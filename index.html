Technologies

I have experience with the following technologies and languages

.NET Framework (C#)

I have primarily worked with Maui .NET to make Android and Windows applications. Fullstack development is made easy and quick with the Maui XAML structure. I have also created Windows applications with the .NET Xamarin.Forms framework, but has since been superceded by Maui.

SQL

No application can quite be complete without SQL. I have worked primarily with SQLite and which houses the database locally. I have also used PostgreSQL and MySQL for server-based databases. While these 3 are relational databases, I have worked with NoSQL which can be non-relational.

Full-Stack Web Development

Using HTML, CSS, and JavaScript, I have created many webpages mostly for accessibility. Given that almost every device can connect to the internet, proper responsive web development has become incredibly important.

Android Development

While Java is becoming less used than Kotlin, they are both pillars in the Android development scene. My first applications were originally made in Java with the Android Studio IDE. In the last several years, Kotlin has become increasingly popular and my most recent applications have been made in Kotlin.

C++

I have created several programs with C++. These have mostly been Command Line Interface (CLI) applications that do not require front end development.

Python

With Python becoming more popular due to its speed and accessibility (needing only an interpreter), I have created mostly scripts for automation and encryption testing.
